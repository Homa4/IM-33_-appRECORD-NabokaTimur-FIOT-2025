Мета лабораторної роботи

Придбати практичні навички роботи верстки сторінок засобами CSS, верстки на основі плаваючих елементів, з’ясувати переваги та недоліки типів макетів веб-сторінок. Придбати практичні навички роботи верстки сторінок засобами CSS та FLEXBOX

Висновки

У ході виконання лабораторної роботи №3 було опановано застосування CSS для оформлення вебсторінок і управління їх візуальною структурою. Під час роботи реалізовано стилізацію елементів інтерфейсу, використано селектори, шрифти, кольори та адаптивне компонування. Отримані результати підтвердили важливість CSS у створенні сучасних, зручних та естетичних веб-застосунків.
